⚙️ Connect to Supabase

This only needs to be done once. After saving, this screen won't appear again.

Go to supabase.com and create a free account, then click New project.
In your project, go to SQL Editor and run this query to create the table:

CREATE TABLE bookings ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, week_key text NOT NULL, cell_id text NOT NULL, teacher text NOT NULL, section text, color text DEFAULT 'blue', created_at timestamptz DEFAULT now(), UNIQUE(week_key, cell_id) );
Go to Project Settings → API. Copy your Project URL and anon / public key and paste them below.
Looks like: https://abcdefghijkl.supabase.co
Long string starting with "eyJ…" — use the anon/public key, not the service key.
